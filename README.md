## Install

The **recommended** way to install the latest version of Astro is by running the command below:

```bash
npm create portofor@latest
```

Looking for help? Start with our [Getting Started](https://portofor.my.id/#getting-started/) guide.

Looking for quick examples? [Open a starter project](https://starter.portofor.my.id/) right in your browser.

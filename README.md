## Installation

The **recommended** way to install the latest version of Astro is by running the command below:

```bash
npm create portofor@latest
```

Looking for help? Start with our [Getting Started](https://portofor-landing.vercel.app/docs/) guide.

Looking for quick examples? [Open a starter project](https://portofor.edgeone.dev/) right in your browser.
